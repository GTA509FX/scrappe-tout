## 🛠 About scrappe-tout

scrappe-tout is a tool designed to turn HTML documentation into neat, easy-to-read Markdown files. It works by following navigation menus, getting rid of repeated information, and naming files clearly so you can find them fast.

You don’t need any programming skills to use it. The tool runs from your computer and helps when you want clean notes or want to move docs to another format.

Key features include:

- Converts HTML pages into Markdown format
- Automatically manages navigation menus to get all pages
- Removes duplicate texts to avoid clutter
- Creates descriptive filenames for easy searching
- Works with many web documentation sites
- Built with lightweight JavaScript and Playwright for smooth scraping

scrappe-tout is ideal if you want well-organized Markdown notes from website docs without manual copy-pasting.

## 🔧 Settings and Options

scrappe-tout provides settings to customize how the scraping works:

- **Link Depth:** Set how many levels deep the tool follows links in menus.
- **Exclude URLs:** Add web addresses you don’t want to scrape.
- **Filename Style:** Choose between full titles or short names for files.
- **Markdown Style:** Pick between standard Markdown or variants used by popular editors.
- **Navigation Handling:** Option to force scraping certain menu types or ignore ads.

You can find these options under the **Settings** tab in the app.

## 🐞 Troubleshooting

You may face some common issues and here are quick fixes:

- **scrappe-tout won’t start:** Try restarting your computer or reinstalling the tool.
- **Download stops or errors occur:** Check your internet connection.
- **Scraping misses pages:** Ensure the URL is a documentation homepage or has a clear menu to follow.
- **Files look messy or have errors:** Try lowering the link depth or checking exclusion rules in Settings.

If problems continue, you can ask for help by opening an issue on scrappe-tout’s GitHub page.
